== Description ==

twitter profile widgetWidget. Plugin shows your tweets on Page/Post/Widget area.

This plugin shows your twitter tweets under Sidebar Area (Widget), Post and/or Pages. It has also a feature that automatically refresh latest tweets. Also it has reply option inside widget on mouse over action. Admin panel option available under" Settings -> Ultimate Twitter Profile Widget" to set different color combination and twitter widget options.

== Installation ==
1. Unpack the `download-package`.
2. Upload the file to the `/wp-content/plugins/` directory.
3. Activate the plugin through the `Plugins` menu in WordPress.
4. Configure the options under Admin Panel `Settings -> Ultimate twitter profile widget`.
5. Done and Ready.
