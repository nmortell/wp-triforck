== Description ==

Youtube Carousel gallery : Youtube carousel gallery uses You tube api to Fetch You tube video Thumb and make carousel from them.On click on those thumbs video plays in prettyphoto.

Installation Guide:
 
1.Upload the  plugin to your blog and just activate it.

2.Goto your Setting>you tube carousel gallery and then Edit its data as per requirement.

3.Use Tiny mce editor button to add shortcodes to your editor.
