== Description ==

Help users to put advertisment content as widget also head tail of page or head and tail of post. Simply add widget to your sidebar and pud ad content as
HTML or JavaScript code or Flash object.
 
Home page: [Devtech](http://www.devtech.cz/ "Devtech - supports preshashop plugin,wordpress plugin,vpn,b2b,eshop,blog,annonce,link,seo,proxy,mailing,affiliate")

== Installation ==

1. Upload plugin folder `ads-content` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Setup your ad widget through the 'Themes/Widgets' menu in WordPress
