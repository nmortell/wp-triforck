Amazing Youtube Player adds automatically a youtube player at the end of pages with youtube videos or push it with selected videos at any post or page.

== Description ==


This plugin adds a friendly and advanced youtube videos player HTML5 compliant (no Flash required). There are two options, you can allow to embed the player automatically at the end of your web pages with youtube videos (also you can add a text before and after the player), or if you prefer the plugin can give you a code to insert where you like to push the player selecting the videos you would like to play.Start to play videos with the amazing youtube player!

Compatible with all Wordpress versions (since 2.3 to 3.3). Don't forget to check the config panel.

== Installation ==

1. Upload the plugin to your intallation plugins directory or add it manually
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Optinally modify the settings by default at the Options panel
